# V-22 Osprey Flight Simulator

TypeScript と Three.js を使用して作られた、V-22 オスプレイの機動飛行シミュレーターのベースプロジェクトです。
V-22の最大の特徴である**ティルトローター機構（ヘリコプターモードから固定翼モードへの転換）**と、それに伴う空気力学的な挙動の変化を簡易的に再現しています。
